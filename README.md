# z2x

z2x is vmlinu**x** extractor from vmlinu**z**.

## Usage

There is no option, so simple.

```
$ z2x vmlinuz-3.10.0-514.6.1.el7.x86_64
```

If vmlinux exists,

```
$ z2x vmlinuz-3.10.0-514.6.1.el7.x86_64
vmlinux is already exist
```

## Installation

You can use `go get`.

```
go get -u github.com/zuiurs/z2x
```
